# Neyman-Pearson Lemma and Its Applications in Machine Learning

This repository contains my internship project report on the **Neyman-Pearson Lemma** and its applications in machine learning. The project focuses on the statistical foundation of hypothesis testing, the role of Type-I and Type-II errors, likelihood-ratio testing, and the connection between the Neyman-Pearson framework and binary classification.

## Project Overview

The Neyman-Pearson Lemma is a classical result in mathematical statistics that gives the most powerful test for a fixed significance level. In this project, the lemma is studied in a clear and structured manner, and its relevance to machine learning is discussed through decision thresholds, asymmetric error control, and classification problems.

The report also includes:
- formal statistical background
- Type-I and Type-II error interpretation
- the Neyman-Pearson decision rule
- threshold trade-offs and ROC intuition
- machine learning connections
- concrete application areas such as medical diagnosis, fraud detection, intrusion detection, and anomaly detection
- research relevance and future scope

## Key Topics Covered

- Hypothesis testing
- Type-I error and Type-II error
- Neyman-Pearson Lemma
- Likelihood-ratio tests
- Most powerful tests
- Decision thresholds
- ROC and threshold trade-off
- Binary classification in machine learning
- Cost-sensitive learning
- Applications in high-stakes decision making

## Objective

The main objective of this project is to present the Neyman-Pearson framework in a simple but mathematically sound way and show why it is useful in modern machine learning problems where different errors have different costs.

## Summary

This work bridges classical statistical hypothesis testing with modern classification-based machine learning. It highlights how the Neyman-Pearson Lemma provides a principled way to control one type of error while optimizing performance under that constraint.
